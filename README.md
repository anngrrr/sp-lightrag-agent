# SP LightRAG Agent

Минималистичный русскоязычный AI-помощник для архитекторов и проектировщиков.
Проект отвечает на вопросы по строительным нормам на основе вашего локального индекса LightRAG.

## Для чего проект
- Быстро получить ориентир по требованиям СП/нормативов.
- Увидеть не только ответ модели, но и контекст, на котором он основан:
  - `Grounding summary`
  - `Reference documents`
  - `Key entities`
  - `Key relationships`
  - `Retrieved chunks`

## Как это работает
- LightRAG использует локальное хранилище графа и чанков (`data/lightrag_storage`).
- Приложение выполняет нативный `query_llm` LightRAG.
- Интерфейс на NiceGUI показывает ответ и ключевой найденный контекст.

## Быстрый старт
1. Установить зависимости:
```bash
uv sync
```
2. Проверить, что настроены переменные окружения (LLM/Embedding/Ollama).
3. Запустить приложение:
```bash
uv run python main.py
```
4. Открыть UI в браузере (`http://localhost:8080`).

## Как пользоваться
1. Введите вопрос на русском языке (например, по эвакуации, ширине проходов, классам пожарной опасности).
2. Прочитайте ответ модели.
3. Сверьте ответ с блоком `Grounding summary` и раскрывающимися секциями контекста.

## Для кого
- Архитекторы
- Проектировщики
- Специалисты по пожарной безопасности

## Ограничения
- Это инструмент поддержки принятия решений, а не юридически значимый автоматический эксперт.
- Финальную проверку норм необходимо делать по актуальным официальным документам.
